Calibration thresholds

      
Effective thresholds for this module. These control how scores are routed to pass, manual review (yellow), or fail.


      
Total

      

        

          
          
          
Minimum total score to pass. Composed of: submission score (0–70) + multiple choice score (0–30). Both sub-gates below must also be met.

        

      


      
Manual review zone

      

        

          
          
          
Scores at or above this value enter the yellow/manual-review zone.

        

        

          
          
          
Scores at or below this value stay in the yellow zone. Must be ≤ pass score. Scores above this pass (if sub-gates are met).

        

      


      
Practical submission

      

        

          
          
          
Minimum percentage of the submission maximum that the practical component must reach. AND-condition — only evaluated for modules with a scored submission component. Skipped automatically if the module has no submission scoring.

        

      


      
Multiple choice questions

      

        

          
          
          
Minimum percentage correct on the multiple choice questions. AND-condition — a submission fails if the MCQ is below this threshold even if the total score passes.
