# Rapportleser

Rapportleseren bruker resultatområdet til å følge utvikling i moduler og kurs.

## Hovedoppgaver

- laste rapportdata med filtre for modul, kurs og dato
- sammenligne aggregert utvikling for moduler og kurs
- åpne detaljrader for å se enkeltbrukere, status og skår

## Arbeidsflate

Rapportleseren jobber primært i [Resultat](/results).

## Greit å vite

- filtrene øverst styrer både sammendrag og detaljvisninger
- rapportleseren endrer ikke innhold eller vurderingsregler
- eksport på deltakernivå spores som eget videre arbeid
